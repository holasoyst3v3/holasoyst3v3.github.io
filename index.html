Toys N' Stuff

Toys N' Stuff Web App

Click for Github Repo

For my final solo capstone project, I decided to keep the parenting theme going and created a used goods platform dedicated towards kids toys and stuff. I also wanted to try out react rather than vanilla JS. Let's be honest, kids can be expensive and you want to spoil them with what you didn't have growing up. There is no shame in getting previously owned goods, in good condition. To our children, it's new to them.

Tech used: - Axios - BCrypt - Cors - Express - Formik - PG - P H-store - React Router - React Router Dom - Sequelize
